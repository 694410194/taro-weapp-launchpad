# Contributing

## Quick Start

1. Fork the repo and create your branch from `main`.
2. Install deps: `pnpm i`
3. Run dev (WeChat): `pnpm dev:weapp`
4. Make changes + add tests/docs when needed.
5. Ensure checks pass: `pnpm check`
6. Open a PR.

## Commit Convention

This repo uses Conventional Commits:

- `feat:` new feature
- `fix:` bug fix
- `docs:` docs only
- `refactor:` refactor without behavior change
- `chore:` tooling / deps
- `test:` tests

## Pull Request Checklist

- [ ] Linked issue / described motivation clearly
- [ ] Added or updated docs (if needed)
- [ ] `pnpm check` / `pnpm format:check` passing
